## Criando um Projeto em NextJS
Repositório destinado a conhecer os procedimentos de criação de um projeto com React/Next-JS.
Nesse projeto, pude aprender alguns comandos para Criação do projeto bem como para sua Execução e também tive uma breve experiência com o gerenciamento de dependências:
- Precisei reinstalar o Next para sua versão 15 devido a incompatibilidades da v.16 com seu novo recursos - Turbopack (que nesse momento, vem apresentando alguma incompatibilidade com o Windows).
