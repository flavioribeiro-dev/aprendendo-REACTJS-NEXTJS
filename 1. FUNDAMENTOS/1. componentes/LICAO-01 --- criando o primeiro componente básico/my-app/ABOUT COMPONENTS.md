# Repositório destinado a minha aprendizagem sobre a criação de componentes básicos.

### Extensão dos arquivos JSX
- Os arquivos podem ter as seguintes extensões: **TSX** (HTML, CSS e Typescript) ou **JSX** (HTML, CSS e Javascript).
    - Como pretendo utilizar os recursos do Typescript, vou optar por usar o padrão de arquivos.TSX

---
### Sobre a criação de Páginas como Componentes
- Cada página funciona como um componente.
- A estrutura básica de cada Componente é:
```
export default function Nome_componente() {
    return (
        <... Tags HTML ...>
    )
}
```
